## Crawling 사용법
----
<br>

### news_crawling.py

- to download news in naver
```bash
press_list : 뉴스 회사 종류 선택
query : 검색할 뉴스 제목
news_num : 검색할 뉴스의 개수
code : python news_crawling.py
```

### google_img_crawling.py

- to download image in google with advanced filter
```bash
python google_img_crawling.py --query [query] --goal_cnt [cnt]
```
